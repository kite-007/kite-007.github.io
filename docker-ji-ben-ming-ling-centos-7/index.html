# Docker基本命令 （CentOS-7）

2021-03-26

  • 1. 初识Docker
  • 2. Docker架构
  • 3. 配置Docker镜像加速
  • 4. Docker命令
(Docker)

1. 初识Docker

  1. Docker概述：
    1. docker为一个开源的应用容器引擎，基于Go语言。
    2. 可以让使用者打包应用以及依赖包到另外一个轻型、可移植的容器中，然后发布到任何流行的linux机器上。
    3. 沙箱机制，相互隔离。
    4. 容器性能开销低。
    5. IO CE EE。
  2. Docker安装：
    1. 升级软件包到最新：
      • yum update
    2. 安装需要升级的软件包，yum-util 提供yum-config-manager功能，另外两个是devicemapper驱动依赖的
      • yum install -y yum-utils device-mapper-persistent-data lvm2
    3. 设置yum源
      • sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    4. 安装Docker
      • yum install -y docker-ce
    5. 查看Docker的版本
      • docker -v
    6. kali中命令docker后选择y安装即可，不过是io版本

2. Docker架构

  1. Clients：客户端
    1. 操作Docker的客户端
  2. Hosts：Docker核心
    1. localhost：本机机器
      1. 在本机机器上面安装完Docker之后，Docker会以daemon(守护进程的形式存在，后台运行)，Docker中有两部分内容，一个叫做image镜像（相当于一个root文件系统），一个container容器，相当于面向对象里的类与对象的关系，镜像来源于远程的仓库。
    2. remotehost：远程机器
  3. Registries：仓库
    1. 官方提供的仓库，存在相当多的镜像文件，可以从本地拉取使用，因为官方在国外，速度比较慢，所以可以搭建私有的仓库。
    2. 私有仓库可以进行资源的共享，速度比较快。

3. 配置Docker镜像加速

  • 默认的情况，会从Docker hub (https://hub.docker.com/)上面下载docker镜像，但是比较慢，一般会配置镜像加速器：
    • USTC、阿里云、网易云、腾讯云等等
# 创建一个docker文件夹
sudo mkdir -p /etc/docker
# 将此配置写入（固定写法）
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://ktem79m6.mirror.aliyuncs.com"]
}
EOF
# 重新加载docker进程
sudo systemctl daemon-reload
# 重启docker服务
sudo systemctl restart docker
  • 其他的镜像加速器修改一下地址即可

4. Docker命令

  1. 操作Docker整体，服务相关命令，开启与关闭Docker命令
# 开启docker服务
systemctl start docker

# 查看docker信息
systemctl status docker

# 停止docker服务
systemctl stop docker

# 重启docker服务
systemctl restart docker

# 设置开机docker自启动
systemctl enable docker
  1. 操作image镜像
# 1. 查看镜像
docker images

#所有镜像id表
docker images -q 

# 2. 搜索镜像
docker search php

# 官方中搜索查找版本信息
hub.docker.com

# 3. 拉取镜像
docker pull php:7.5

# 4. 删除镜像
docker rei REPOSITORY:TAG
docker rei IMAGE_ID
id相同表示相同版本

#删除所有镜像,这里的引号为tab键上面的引号
docker rei `docker images -q`
  1. 操作容器
    1. 查看容器
    docker ps 
# 查看开启的容器

docker ps -a
# 查看所有的容器，包括历史打开过的
    1. 创建并启动容器
    docker run -it --name=c1 centos:7 /bin/bash
# i：表示容器一直运行着，无客户端连接也会运行
# t：给容器分配一个终端输入命令
# --name：给容器起个名字
# centos:7：镜像版本
# /bin/bash：进入容器初始化指令，打开一个shell窗口

exit退出之后容器会自动关闭

docker run -id --name=c2 centos:7 /bin/bash
# d：表示后台运行创建容器，可以通过命令进入容器，

exit退出之后容器不会自动关闭
    1. 进入容器
    docker exec -it c2 /bin/bash
    1. 启动容器
    docker start c2
    1. 停止容器
    docker stop c2
    1. 删除容器
    docker rm c1
docker rm `docker ps -aq`
# 开启的容器不能被删除
    1. 查看容器信息
    docker inspect c1
